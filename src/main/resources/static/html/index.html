如何判断代码的好坏


看一段代码，无需看函数签名、参数类型等，直接看代码的命名、执行流程，就可以知道该段代码的含义，即表示该代码是好代码。 工作中遇到一时没看懂的代码，会去看函数签名、去看函数的内部逻辑，虽然最后都看懂了，但却花费了更多时间与精力。

整洁的代码，可以更快的发现问题。
